# Тайна Темного Лабиринта 🏰

Привет! Это мой проект — небольшая игра "Тайна Темного Лабиринта", которую я делаю сам. 
Вся игра написана на Python. Сейчас я активно экспериментирую с 3D-системой (псевдо-3D в стиле старых игр), добавляю новые текстуры, звуки и улучшаю код.

## Что тут есть:
- Исходный код игры (папка `Code`).
- Всякие разные текстуры для лабиринта (стены, пол, двери).
- Звуки и музыка.
- Простенький редактор уровней (`editor3000.py`), который я сделал для удобства.

## Как запустить:
Так как проект пока в активной разработке, готового `.exe` файла нет. Для запуска нужен установленный Python и, скорее всего, библиотека Pygame.
Просто скачай проект, открой папку `Code` и запусти самый свежий файл с игрой (например, `Код 7.0 Но система 3d 3.0.py`).
